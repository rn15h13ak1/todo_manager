# Changelog

## [1.15.1] - 2026-04-16

### Fixed
- ステータス・優先度・期限の簡易変更ポップオーバー表示中に ESC キーを押すと変更せずに閉じるよう修正
- ポップオーバー表示中の ESC キーが App のフィルターリセット処理と競合しないよう修正

## [1.15.0] - 2026-04-16

### Added
- タスクカードのタイトル・説明・余白など任意の箇所をクリックすると編集モーダルを開けるようにした
- カードホバー時に薄いグレー背景を表示してクリック可能であることを視覚的に示すようにした
- ステータス・優先度・期限・タグ・チェックボックス・削除ボタンは引き続きそれぞれ独立して動作するよう維持

## [1.14.0] - 2026-04-16

### Added
- タスクカードの期限テキストをクリックするとポップオーバーが表示され、日付入力で期限を設定・変更できる機能を追加
- 期限が設定されている場合は「削除」ボタンで期限を削除できる機能を追加
- 「期限なし」のタスクもクリックで期限を新規設定できるように対応

## [1.13.0] - 2026-04-16

### Added
- ステータス・優先度を簡易変更した際、タスクの表示位置が変わった場合にスムーズスクロールで移動先を追従する機能を追加
- 変更後のタスクカードを黄色でフラッシュハイライトし、移動先を視覚的に確認できるようにした

## [1.12.0] - 2026-04-16

### Added
- タスクカードの優先度バッジをクリックするとポップオーバーが表示され、モーダルを開かずに優先度を即座に変更できる機能を追加

## [1.11.0] - 2026-04-16

### Added
- タスクカードのステータスバッジをクリックするとポップオーバーが表示され、モーダルを開かずにステータスを即座に変更できる機能を追加

## [1.10.3] - 2026-04-16

### Fixed
- 期限日ソート時、同じ期限日のタスクを優先度（高→中→低）で第2ソートするよう修正

## [1.10.2] - 2026-04-16

### Fixed
- 期限切れ判定の「今日」が UTC 日付になっていたバグを修正（ローカル時刻ベースに変更）

## [1.10.1] - 2026-04-16

### Fixed
- `n` キーでモーダルを開く際、タイトル入力欄に「n」が入力されてしまうバグを修正

## [1.10.0] - 2026-04-16

### Added
- `n` キーでタスク追加モーダルを開くショートカットを追加（入力中・モーダル表示中は無効）

## [1.9.1] - 2026-04-16

### Fixed
- タスク追加・編集モーダル表示中に ESC キーを押すとキャンセル扱いでモーダルを閉じるよう修正

## [1.9.0] - 2026-04-16

### Added
- ESC キーでフィルターを初期状態にリセットする機能を追加（モーダル表示中は無効）

## [1.8.0] - 2026-04-16

### Added
- ヘッダーにハンバーガーメニューを追加
  - JSONインポート・JSONエクスポート・HTMLエクスポートをメニュー内に移動
  - 画面下部のインポート/エクスポートボタンを削除

## [1.7.0] - 2026-04-16

### Added
- ヘッダーとフィルターバーをスクロール時も常時表示（sticky固定）に変更

## [1.6.0] - 2026-04-16

### Added
- フィルターリセット機能を追加
  - 初期状態から変更がある場合のみ「✕ フィルターをリセット」ボタンを表示
  - クリックで全フィルター・ソートを初期状態（ステータス：完了以外、期限日昇順）に一括リセット

## [1.5.0] - 2026-04-16

### Added
- タスクカードのタグバッジをクリックすると、そのタグでフィルターバーを自動切替えする機能を追加

## [1.4.1] - 2026-04-16

### Fixed
- 一部選択中（indeterminate）に「全て選択」チェックボックスを1クリックで全解除できるよう修正
- 選択中に「× 解除」ボタンを追加し、明示的に選択をクリア可能に

## [1.4.0] - 2026-04-16

### Added
- 一括タグ追加・削除に確認ステップを追加
  - タグ選択後にモーダル内で確認画面を表示（タグ名・対象件数）
  - 「戻る」で選択画面に戻る、「追加する / 削除する」で実行

## [1.3.1] - 2026-04-15

### Fixed
- 日本語IME変換確定のEnterキーでタグが追加されるバグを修正（`isComposing` チェックを追加）

## [1.3.0] - 2026-04-15

### Added
- タグ機能
  - タスクへのタグ追加・削除（テキスト入力 + Enter、または既存タグからサジェスト選択）
  - タグごとに一定の配色（ハッシュベース）でバッジ表示
  - タグによるフィルター（フィルターバーに「タグ」セレクト追加）
  - 複数タスクへの一括タグ追加・削除（バルクタグモーダル）

## [1.2.0] - 2026-04-15

### Added
- 複数タスクの一括削除機能
  - 各タスクカードにチェックボックスを追加
  - 「全て選択」チェックボックス（indeterminate 対応）
  - 選択中の件数表示と「X 件を削除」ボタン
  - 削除前の確認ダイアログ

## [1.1.0] - 2026-04-15

### Added
- ステータスフィルターに「完了以外」を追加（未着手・進行中をまとめて絞り込み）

## [1.0.0] - 2026-04-15

### Added
- タスクの CRUD（追加・編集・削除）
- タスクデータ構造：タイトル・説明・期限日・優先度・ステータス・作成日時
- ステータスフィルター（すべて / 未着手 / 進行中 / 完了）
- 優先度フィルター（すべて / 高 / 中 / 低）
- 期限切れフィルター（今日以前の未完了タスク）
- ソート（期限日昇順・降順 / 優先度 / 作成日）
- 期限切れタスクの視覚的強調（赤ボーダー・赤テキスト）
- localStorage による自動保存・自動読み込み
- JSON インポート（.json ファイルからタスクを復元）
- JSON エクスポート（`todo-manager_yyyymmdd_hhmmss.json`）
- HTML エクスポート（`todo-manager_yyyymmdd_hhmmss.html`）
- 削除時の確認ダイアログ（タスク名入り）
- 編集・削除ボタンの間隔調整（誤クリック防止）
- ヘッダーへのバージョン表示
- Vite + vite-plugin-singlefile による単一 HTML ファイルビルド
