# Changelog

## [1.26.0] - 2026-04-16

### Added
- フィルター条件のプリセット保存・呼び出し機能を追加
  - 「☆ プリセット」ボタンで保存済みプリセット一覧を表示
  - 現在のフィルター・ソート条件を名前付きで保存
  - プリセットをクリックするとフィルター条件を即座に適用
  - 不要なプリセットはホバー時に表示される ✕ ボタンで削除可能
  - プリセットは localStorage に永続保存

## [1.25.0] - 2026-04-16

### Added
- 完了タスクをアーカイブセクションとして折りたたむ機能を追加
  - 完了タスクはリスト下部の「完了済み（N件）」セクションにまとめて表示
  - クリックで展開・折りたたみが可能
  - 展開時は半透明（opacity-70）で通常タスクと区別しやすくした

## [1.24.0] - 2026-04-16

### Added
- タスクリストにコンパクト表示モードを追加
  - 「コンパクト」ボタンで表示密度を切り替え（説明・タグ・期限を非表示）
  - コンパクト時はカード間の余白も縮小し、より多くのタスクを一覧表示できる
  - 「通常」ボタンで元の表示に戻せる

## [1.23.0] - 2026-04-16

### Added
- ヘッダーにステータス別タスク件数サマリーを追加（未着手・進行中・完了の件数をリアルタイム表示）

## [1.22.0] - 2026-04-16

### Added
- 期限日を相対表示に変更（「今日」「昨日」「明日」「N日前/後」「N週間前/後」「Nヶ月前/後」）
  - 絶対日付 (YYYY-MM-DD) も括弧内に併記して常に確認できるようにした
  - `src/utils/date.js` に `formatRelativeDate` ユーティリティを追加

## [1.21.0] - 2026-04-16

### Added
- `c` キーでフォーカス中タスクの完了状態をワンクリック切り替えする機能を追加（完了 ↔ 未着手）
- ショートカット一覧モーダル（`?`）に `c` キーの説明を追加

## [1.20.0] - 2026-04-16

### Added
- `?` キーでキーボードショートカット一覧モーダルを表示する機能を追加
  - 全般・タスク操作・フィルター操作の3セクションで全ショートカットを一覧表示
  - `Esc` キーまたは `✕` ボタン、オーバーレイクリックで閉じられる
  - 入力中・モーダル表示中は `?` キーを無効化

## [1.19.0] - 2026-04-16

### Added
- タイトル・説明文によるテキスト検索機能を追加
  - フィルターバーに検索欄を追加（タイトル・説明を対象に部分一致で絞り込み）
  - `/` キーで検索欄にフォーカスするキーボードショートカットを追加
  - 検索中はフィルターリセットボタンを表示し、ESC キーでリセット可能

## [1.18.0] - 2026-04-16

### Added
- `f` キーでフィルター項目にフォーカスを当てる機能を追加（ステータス→優先度→タグ→ソート→期限切れのみ→ステータスの順で循環）
- フィルターフォーカス中に `j`/`k` キーでドロップリストの選択肢を上下に切り替えられるようにした
- フィルターフォーカス中に `Space` キーで「期限切れのみ」チェックボックスをトグルできるようにした
- フィルターフォーカス中の `j`/`k`/`Space` がタスク操作と干渉しないよう制御を追加
- `ESC` の処理順序にフィルターフォーカス解除を先頭に追加

## [1.17.1] - 2026-04-16

### Fixed
- dキーで一括削除する際の確認ダイアログが「0件を削除しますか？」と表示される不具合を修正
  - 登録済み関数が古いクロージャを参照していた問題を ref パターンで解消

## [1.17.0] - 2026-04-16

### Added
- `d` キーで選択中タスクの一括削除（チェックボックスがオンのとき）または、フォーカス中タスクの削除（チェックボックスがオフでフォーカスがあるとき）ができるようにした
- いずれも確認ダイアログを表示してから削除するようにした

## [1.16.3] - 2026-04-16

## [1.16.2] - 2026-04-16

## [1.16.1] - 2026-04-16

## [1.16.0] - 2026-04-16

## [1.15.1] - 2026-04-16

## [1.0.0] - 2026-04-15

### Added
- タスクの CRUD（追加・編集・削除）
- タスクデータ構造：タイトル・説明・期限日・優先度・ステータス・作成日時
- ステータスフィルター（すべて / 未着手 / 進行中 / 完了）
- 優先度フィルター（すべて / 高 / 中 / 低）
- 期限切れフィルター（今日以前の未完了タスク）
- ソート（期限日昇順・降順 / 優先度 / 作成日）
- 期限切れタスクの視覚的強調（赤ボーダー・赤テキスト）
- localStorage による自動保存・自動読み込み
- JSON インポート（.json ファイルからタスクを復元）
- JSON エクスポート（`todo-manager_yyyymmdd_hhmmss.json`）
- HTML エクスポート（`todo-manager_yyyymmdd_hhmmss.html`）
- 削除時の確認ダイアログ（タスク名入り）
- 編集・削除ボタンの間隔調整（誤クリック防止）
- ヘッダーへのバージョン表示
- Vite + vite-plugin-singlefile による単一 HTML ファイルビルド
