# todo-manager

ブラウザで動作するシンプルな TODO 管理ツールです。
ビルド成果物は単一の HTML ファイル（`dist/todo-manager.html`）として出力されるため、サーバー不要でどこでも使えます。

## 機能

### タスク管理
- タスクの追加・編集・削除
- タスク情報：タイトル・説明・期限日・優先度（高/中/低）・ステータス（未着手/進行中/完了）・タグ
- 削除時の確認ダイアログ
- 期限切れタスクの視覚的強調（赤ボーダー・赤テキスト）

### フィルター・ソート
| 項目 | 選択肢 |
|------|--------|
| ステータス | すべて / 完了以外 / 未着手 / 進行中 / 完了 |
| 優先度 | すべて / 高 / 中 / 低 |
| タグ | すべて / 各タグ |
| 期限切れのみ | チェックボックス |
| ソート | 期限日昇順・降順 / 優先度 / 作成日（新しい順）|

- フィルターバーの「✕ フィルターをリセット」で初期状態に一括リセット

### タグ
- タスクごとにタグを複数設定
- テキスト入力（Enter）または既存タグからサジェスト選択
- ハッシュベースの配色バッジ表示
- タグバッジをクリックするとそのタグでフィルター
- 複数タスクへの一括タグ追加・削除（確認ステップあり）

### 一括操作
- チェックボックスで複数タスクを選択
- 一括削除（確認ダイアログあり）
- 一括タグ追加・削除

### データ管理（ハンバーガーメニュー）
- **JSON インポート**：`.json` ファイルからタスクを復元
- **JSON エクスポート**：`todo-manager_yyyymmdd_hhmmss.json` として保存
- **HTML エクスポート**：`todo-manager_yyyymmdd_hhmmss.html` として保存
- **localStorage** による自動保存・自動読み込み

### キーボードショートカット
| キー | 動作 |
|------|------|
| `n` | タスク追加モーダルを開く |
| `ESC` | モーダルを閉じる（キャンセル）/ フィルターをリセット |

### UI
- ヘッダー・フィルターバーのスクロール時常時表示（sticky）
- ヘッダーのバージョン番号をクリックで変更履歴を表示

## 開発

### 必要環境
- Node.js 18 以上

### セットアップ
```bash
npm install
```

### 開発サーバー起動
```bash
npm run dev
```

### ビルド
```bash
npm run build
# → dist/todo-manager.html が生成されます
```

### プレビュー
```bash
npm run preview
# → http://localhost:4173/todo-manager.html
```
